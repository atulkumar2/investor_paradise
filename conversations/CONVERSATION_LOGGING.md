# Conversation History Tracking

This document has moved into the `conversations/` folder. See also `CONVERSATION_LOGGING_QUICKSTART.md`.

## Overview

The Investor Paradise agent includes comprehensive conversation logging with session tracking. All user queries and agent responses are logged to `conversations/conversation_history.jsonl` with session IDs for easy tracking and analysis.

## Quick Links

- View all sessions: `python conversations/view_conversations.py`
- Export ADK web sessions: `python conversations/export_adk_sessions.py`
- Quickstart: `CONVERSATION_LOGGING_QUICKSTART.md`
